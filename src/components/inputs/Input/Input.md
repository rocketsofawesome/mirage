## Regular Styles

#### Default input
```js
  <Input label='Email' />
```

#### Masked input
```js
  <Input label='Name' type='text' mask='(999) 999 9999' />
```

#### Input with errors
```js
  <Input label='Email' error='This is some error message' touched />
```
